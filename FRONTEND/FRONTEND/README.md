～フロントエンド～
概要

このプロジェクトは、音声ファイルをアップロードして文字起こしと要約を行う Next.js（React）フロントエンドです。
FastAPI サーバーに音声ファイルを送信し、文字起こし・要約・重要ポイントを取得して表示します。
Tailwind CSS を用いたシンプルでモダンなUIとなっています。

主な機能

・ファイル選択またはドラッグ＆ドロップによる音声アップロード
・アップロード可能形式：.wav, .mp3, .m4a など
・サーバーに送信して文字起こし・要約・重要ポイントを取得
・文字起こし結果と要約を画面に表示
・処理中はローディングアニメーションを表示
・「もう一度処理する」でファイルをリセット可能
・エラー時にはアラートで通知

実装の特徴・利用構文

・useState でファイル・ロード状態・結果の管理
・onChange と onDrop でファイル選択とドラッグ＆ドロップに対応
・fetch と FormData を用いた FastAPI への POST リクエスト
・async/await と try/catch/finally で非同期処理とエラー制御
・条件付きレンダリング ({result ? ... : ...}) による UI 表示切替
・Tailwind CSS によるレスポンシブかつモダンなデザイン
・whitespace-pre-wrap と overflow-y-auto を利用した文字起こしの表示
・ボタンの disabled 属性で操作中の多重リクエストを防止
・配列の map 関数で重要ポイントを番号付きリストとして表示
